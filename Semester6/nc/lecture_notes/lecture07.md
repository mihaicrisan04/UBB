# Lecture 7 — Birkhoff Interpolation, Peano's Theorem, and Spline Interpolation (Linear Splines)

## Overview
This lecture introduces **Birkhoff (lacunary Hermite) interpolation**, the most general polynomial interpolation problem, where only some functional or derivative values are prescribed at the nodes. It then develops **Peano's theorem** to express the interpolation error via the Peano kernel, and finally motivates and defines **spline interpolation**, working out the **linear spline** case in detail.

## Key Notions & Definitions

- **Birkhoff (lacunary Hermite) interpolation problem** — Given $m+1$ distinct nodes $x_k \in [a,b]$, $k = \overline{0,m}$, index sets $I_k \subseteq \{0,1,\dots,r_k\}$, and a function $f$ whose derivatives $f^{(j)}(x_k)$ exist, find a polynomial $P(x)$ of minimum degree satisfying $$P^{(j)}(x_k) = f^{(j)}(x_k),\quad k = \overline{0,m},\; j \in I_k.$$ More general than Hermite: only *some* of the functional/derivative values are given (it is "lacunary" because some are missing).

- **Degree of the Birkhoff polynomial** — With $n+1 = |I_0| + \cdots + |I_m|$ interpolation conditions, one seeks $P_n(x) = a_0 + a_1 x + \cdots + a_n x^n$ of degree at most $n$. The coefficients come from the linear system generated by the conditions; if its determinant is nonzero, the solution is unique.

- **Birkhoff polynomial form** — When it exists, the Birkhoff polynomial $B_n f$ has the form $$B_n f(x) = \sum_{k=0}^{m} \sum_{j \in I_k} b_{kj}(x)\, f^{(j)}(x_k).$$

- **Birkhoff fundamental polynomials** $b_{kj}(x)$ — Satisfy $$b_{kj}^{(p)}(x_\nu) = 0,\; \nu \neq k,\; p \in I_\nu,\qquad b_{kj}^{(p)}(x_k) = \delta_{jp},\; p \in I_k,$$ where $\delta_{jp}$ is Kronecker's symbol. In general they are hard to find analytically and are determined directly from these conditions.

- **Space $H^n[a,b]$** — $H^n[a,b] = \{ f : [a,b] \to \mathbb{R} \mid f \in C^{n-1}[a,b],\ f^{(n-1)}$ absolutely continuous on $[a,b]\}$. It is a linear space.

- **Absolutely continuous function** — $f$ has a derivative $f'$ almost everywhere, $f'$ is Lebesgue integrable, and $f(x) = f(a) + \int_a^x f'(t)\,dt$ for all $x \in [a,b]$.

- **Taylor-type representation with integral remainder** — Any $f \in H^n[a,b]$ satisfies $$f(x) = \sum_{k=0}^{n-1} \frac{(x-a)^k}{k!} f^{(k)}(a) + \int_a^x \frac{(x-t)^{n-1}}{(n-1)!} f^{(n)}(t)\,dt.$$

- **Positive part / truncated power** — $z_+ = z$ for $z \ge 0$ and $0$ for $z < 0$; $z_+^n = (z_+)^n$ is the truncated power.

- **Kernel of a linear map** — $\ker L = \{ v \in V \mid L(v) = \mathbf{0}_W \}$.

- **Degree of precision (exactness)** — An approximation formula $f = B_n f + R_n f$ has degree of precision $d$ if $R_n e_k = e_k - B_n e_k \equiv 0$ for all $k \le d$ (exact for all polynomials up to degree $d$), where $e_k(x) = x^k$.

- **Splines** — Piecewise polynomials on a grid that satisfy continuity conditions for the function and some of its derivatives. The idea: on a sufficiently small subinterval a function can be approximated arbitrarily well by a low-degree polynomial. Term coined by I. J. Schoenberg (1946); theory developed from 1964.

- **Spline space $\mathbb{S}_m^k(\Delta)$** — For a grid $\Delta : a = x_1 < x_2 < \cdots < x_n = b$, $$\mathbb{S}_m^k(\Delta) = \{ s \mid s \in C^k[a,b],\ s|_{[x_i,x_{i+1}]} \in \mathbb{P}_m,\ i = 1,\dots,n-1 \},$$ the space of polynomial splines of degree $m$ and class $k$. One assumes $0 \le k < m$; for $k = m$, $S_m^m = \mathbb{P}_m$; $k = -1$ allows discontinuities at grid points.

- **Linear spline** — Case $m = 1$, $k = 0$: $s_1 \in \mathbb{S}_1^0(\Delta)$ with $s_1(x_i) = f_i$. On $[x_i, x_{i+1}]$ it is the degree-1 interpolant $$s_1(f;x) = f_i + f[x_i, x_{i+1}](x - x_i) = f_i + \frac{f(x_{i+1}) - f(x_i)}{x_{i+1} - x_i}(x - x_i).$$

- **Linear B-spline (hat) functions** — Basis of $\mathbb{S}_1^0(\Delta)$. With $x_0 = x_1 = a$, $x_{n+1} = x_n = b$, for $i = \overline{1,n}$, $$B_i(x) = \begin{cases} \dfrac{x - x_{i-1}}{x_i - x_{i-1}}, & x_{i-1} \le x \le x_i \\[2mm] \dfrac{x_{i+1} - x}{x_{i+1} - x_i}, & x_i \le x \le x_{i+1} \\[1mm] 0, & \text{otherwise} \end{cases}$$ They satisfy $B_i(x_j) = \delta_{ij}$, are linearly independent, and any $s \in \mathbb{S}_1^0(\Delta)$ is written uniquely as $s(x) = \sum_{i=1}^{n} c_i B_i(x)$. The first and last hats are cut in half.

## Theorems & Results

### Remark 1.1 (Birkhoff generalizes Hermite/Lagrange)
If $I_k = \{0,1,\dots,r_k\}$ for every $k$, the Birkhoff problem reduces to Hermite interpolation; if additionally $r_k = 0$ for all $k$, it reduces to Lagrange interpolation. Hence Birkhoff is the most general.

### Existence is not guaranteed
Unlike Lagrange and Hermite, the Birkhoff problem does **not** always have a solution (the linear system may be incompatible). When the system determinant is nonzero the solution exists and is unique.

### Theorem 1.6 (Peano)
Let $L$ be a real continuous linear functional defined on $H^{n+1}[a,b]$ that commutes with the definite integral operator. If $\ker L = \mathbb{P}_n$, then $$L f(x) = \int_a^b K_n(x,t)\, f^{(n+1)}(t)\,dt,$$ where the **Peano kernel** is $$K_n(x,t) = \frac{1}{n!} L\big( (x - t)_+^n \big).$$
Interpretation: if $L$ annihilates all polynomials of degree $\le n$ (i.e. $L e_k \equiv 0$ for $k = 0,\dots,n$) but $L e_{n+1} \not\equiv 0$, then $Lf$ has the integral representation above. This is paramount in numerical analysis for expressing approximation errors.

### Corollary 1.7 (mean-value form of the error)
If the kernel $K_n$ has constant sign on $[a,b]$ and $f^{(n+1)}$ is continuous, then there exists $\xi \in (a,b)$ with $$L f = \frac{1}{(n+1)!} f^{(n+1)}(\xi)\, L e_{n+1}.$$
**Proof sketch.** Constant sign of $K$ allows the mean value theorem in the integral: $Lf(x) = f^{(n+1)}(\xi)\int_a^b K_n(x,t)\,dt$. The kernel does not depend on $f$, so take $f = e_{n+1}$ to get $\int_a^b K_n\,dt = \frac{1}{(n+1)!} L e_{n+1}$; substitute back.

### Remark 1.8 (error of Birkhoff interpolation)
Apply Peano/Corollary 1.7 to the **remainder functional** $R_n f = f - B_n f$. Since $B_n$ has degree $n$, the formula is exact for degree-$n$ polynomials, so it has degree of precision $d = n$. If the Peano kernel $K_n(x,t) = \frac{1}{n!}\big[(x-t)_+^n - B_n((x-t)_+^n)\big]$ has constant sign on the smallest interval containing the nodes, then $$R_n f = \frac{1}{(n+1)!} f^{(n+1)}(\xi)\, R_n e_{n+1},\quad \xi \in (a,b).$$
Useful derivatives/integrals of the truncated power: $\partial_x[(x-t)_+^n] = n(x-t)_+^{n-1}$ and $\int_a^b (x-t)_+^n\,dx = \frac{1}{n+1}\big[(b-t)_+^{n+1} - (a-t)_+^{n+1}\big]$.

### Linear spline error
For $s_1$, the interpolation error is $$|f(x) - s_1(f;x)| \le \frac{|(x-x_i)(x-x_{i+1})|}{2!}\max_{[x_i,x_{i+1}]}|f''| \le \frac{h_i^2}{8}\max_{[x_i,x_{i+1}]}|f''|,$$ with $h_i = x_{i+1} - x_i$. Globally, with $|\Delta| = \max_i h_i$, $$\|f - s_1(f;\cdot)\|_\infty \le \frac{|\Delta|^2}{8}\|f''\|_\infty.$$

### Dimension of $\mathbb{S}_1^0(\Delta)$
Counting $2(n-1)$ degrees of freedom minus $n-2$ continuity constraints at interior nodes: $$\dim \mathbb{S}_1^0(\Delta) = 2(n-1) - (n-2) = n.$$

## Example Problems

### Example: Birkhoff polynomial from $f(0)=1$, $f'(1)=2$ (Example 1.3)
Nodes $x_0 = 0$, $x_1 = 1$, with $I_0 = \{0\}$, $I_1 = \{1\}$, so $n = 1+1-1 = 1$; seek $P(x) = a_0 + a_1 x$ with $P(0) = f(0)$, $P'(1) = f'(1)$. The system gives $a_0 = f(0)$, $a_1 = f'(1)$, determinant $1 \neq 0$, so $$P(x) = f(0) + f'(1)x = 1 + 2x.$$ Via fundamental polynomials (illustrating the $b_{kj}$ definition): $b_{00}(x) = 1$ and $b_{11}(x) = x$, giving $B_1 f(x) = f(0) + x f'(1)$. This uses the **Birkhoff polynomial form** and the **fundamental polynomial conditions (1.3)**.

### Example: A Birkhoff problem with no solution (Example 1.4)
Find minimum-degree $P$ with $P(-1) = P(1) = 0$, $P'(0) = 1$. Three nodes, $I_0 = \{0\}, I_1 = \{1\}, I_2 = \{0\}$, $n = 2$, so $P(x) = a_0 + a_1 x + a_2 x^2$. The system is
$$\begin{cases} a_0 - a_1 + a_2 = 0 \\ a_1 = 1 \\ a_0 + a_1 + a_2 = 0 \end{cases}$$
Subtracting equation 1 from equation 3 gives $a_1 = 0$, contradicting $a_1 = 1$. The system is incompatible, so **no solution exists** — illustrating that Birkhoff interpolation need not be solvable.

### Example: Error formula via Peano for $B_1 f$ (Example 1.9)
For $B_1 f(x) = f(0) + f'(1)x$ on $[0,1]$, apply Peano to $L = R_1$. Checking $R_1 e_0 \equiv 0$, $R_1 e_1 \equiv 0$, but $R_1 e_2 = x^2 - 2x \not\equiv 0$, so degree of precision $d = 1$. The kernel $K_1(x,t) = (x-t)_+ - (-t)_+ - x = (x-t)_+ - x$ is shown to satisfy $K_1 \le 0$ on $[0,1]$ (constant sign). By **Corollary 1.7**, $$R_1 f(x) = \frac{x^2 - 2x}{2!} f''(\xi),\quad \xi \in (0,1),\qquad |R_1 f(x)| \le \tfrac{1}{2}\|f''\|_\infty.$$ This directly uses **Theorem 1.6 / Corollary 1.7**.

### Example: Quadratic Birkhoff interpolation of $f'(0), f(1), f'(2)$ (Example 1.10)
Nodes $0,1,2$ with $I_0 = \{1\}, I_1 = \{0\}, I_2 = \{1\}$, $n = 2$, $B_2(x) = ax^2 + bx + c$. Conditions $B_2'(0) = f'(0)$, $B_2(1) = f(1)$, $B_2'(2) = f'(2)$ give a system with determinant $4 \neq 0$, so a unique solution exists. The fundamental polynomials are $b_{01}(x) = -\frac{(x-1)(x-3)}{4}$, $b_{10}(x) = 1$, $b_{21}(x) = \frac{x^2-1}{4}$, hence $$B_2 f(x) = \tfrac{1}{4}(x-1)(3-x)f'(0) + f(1) + \tfrac{1}{4}(x^2-1)f'(2).$$
**Part b)** With $f'(0)=1, f(1)=2, f'(2)=1$: $B_2 f(x) = x + 1$, so $f(1/2) \approx B_2 f(1/2) = 3/2$. The remainder check gives degree of precision $d = 2$ ($R_2 e_3 \not\equiv 0$). The Peano kernel $K_2(1/2,t)$ is shown constant-sign (three cases on $t$), so $$R_2 f(1/2) = \frac{1}{3!} f'''(\xi) R_2 e_3(1/2) = \frac{11}{48} f'''(\xi),\qquad |R_2 f(1/2)| \le \frac{11}{48}\|f'''\|_\infty.$$ This combines the **Birkhoff form**, **fundamental polynomial conditions**, and **Peano error (Corollary 1.7)**.

## Cheat-sheet
- Birkhoff conditions: $P^{(j)}(x_k) = f^{(j)}(x_k)$, $j \in I_k$; degree $n$ with $n+1 = \sum_k |I_k|$.
- Birkhoff polynomial: $B_n f = \sum_k \sum_{j \in I_k} b_{kj}(x) f^{(j)}(x_k)$.
- Fundamental polynomial conditions: $b_{kj}^{(p)}(x_\nu) = 0$ ($\nu \neq k$), $b_{kj}^{(p)}(x_k) = \delta_{jp}$.
- Birkhoff may have **no** solution; unique iff system determinant $\neq 0$.
- Truncated power: $z_+^n$ with $\partial_x (x-t)_+^n = n(x-t)_+^{n-1}$.
- Peano: $Lf = \int_a^b K_n(x,t) f^{(n+1)}(t)\,dt$, $K_n = \frac{1}{n!} L((x-t)_+^n)$, when $\ker L = \mathbb{P}_n$.
- Constant-sign kernel $\Rightarrow$ $Lf = \frac{1}{(n+1)!} f^{(n+1)}(\xi) L e_{n+1}$.
- Degree of precision $d$: $R_n e_k \equiv 0$ for $k \le d$.
- Spline space: $\mathbb{S}_m^k(\Delta)$, piecewise $\mathbb{P}_m$, globally $C^k$, $0 \le k < m$.
- Linear spline on $[x_i,x_{i+1}]$: $s_1 = f_i + \frac{f_{i+1}-f_i}{x_{i+1}-x_i}(x-x_i)$.
- Linear spline error: $\|f - s_1\|_\infty \le \frac{|\Delta|^2}{8}\|f''\|_\infty$.
- $\dim \mathbb{S}_1^0(\Delta) = n$; basis = hat (linear B-spline) functions $B_i$ with $B_i(x_j) = \delta_{ij}$.
