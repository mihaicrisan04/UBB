# Bachelor Graduation Exam — Cheatsheets (June/September 2026)

Computer Science specialization. Three subjects, 3 hours, minimum passing grade 5.00, all subjects compulsory.

| File | Course | Exam tasks |
|------|--------|-----------|
| [01-algorithms-and-programming.md](01-algorithms-and-programming.md) | Programming fundamentals, OOP, Data structures & algorithms | implement a function/class (recursive often required), state complexity, implement a UML diagram |
| [02-databases.md](02-databases.md) | Databases | design a BCNF schema, write/evaluate an SQL query, check functional dependencies |
| [03-operating-systems.md](03-operating-systems.md) | Operating systems | trace a C `fork`/`exec`/`wait` program, trace a Bash script (`sed -E`/`grep -E`) |

### Topic coverage map
- **Alg & Prog** — search (sequential/binary), merging, sorting (selection/bubble/insertion/merge/quick), backtracking; pseudocode & complexity; OOP (class/object, access modifiers, constructors/destructors); inheritance, overriding, polymorphism, dynamic binding, abstract classes/interfaces; UML class diagrams & relations; lists, maps, BSTs, hash tables + library equivalents.
- **Databases** — relational model & integrity constraints; SQL DDL/DML; 3-valued logic; full SELECT (joins, set ops, aggregates, subqueries); functional dependencies + Armstrong axioms; normal forms 1NF–BCNF; relational algebra.
- **Operating Systems** — UNIX file system; processes (fork/exec/exit/wait, pipe, FIFO); shell programming (variables, control structures, predefined vars, I/O redirection, ERE) + command reference.

> Languages allowed for programming items: Python, C++, Java, C#, or pseudocode.
