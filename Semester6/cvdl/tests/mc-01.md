## Basic layers in DL

**1. Which activation function outputs values strictly in the range (0, 1)?**
- A) ReLU
- B) Tanh
- C) Sigmoid
- D) Linear

Your answer: ~~A~~ ❌ → **C (Sigmoid)**
> ReLU outputs [0, ∞), so it isn't bounded above. Sigmoid squashes any input into (0, 1). Tanh is (−1, 1). Linear is unbounded.

**2. The main purpose of batch normalization is to:**
- A) Reduce model size
- B) Stabilize and speed up training by normalizing layer inputs across the batch
- C) Replace pooling layers
- D) Eliminate the need for activation functions

Your answer: **B** ✅

**3. Dropout is used during training to:**
- A) Increase model capacity
- B) Speed up the forward pass
- C) Prevent overfitting by randomly zeroing some activations
- D) Replace batch normalization

Your answer: ~~B~~ ❌ → **C**
> Dropout makes training a bit *slower*, not faster. Its purpose is regularization: at each iteration, a random subset of neurons is zeroed out, forcing redundancy and preventing co-adaptation.

**5. VGG architectures are characterized by:**
- A) Asymmetric kernels
- B) Stacking many small 3×3 convolutions in series
- C) Residual connections
- D) Inception modules

Your answer: ~~blank~~ ❌ → **B**
> VGG's signature is "always 3×3 convolutions, always 2×2 max pooling." Two stacked 3×3 convs have the same receptive field as one 5×5 but with fewer parameters and an extra non-linearity.

**10. Mean Squared Error (MSE) is most appropriate for:**
- A) Multi-class classification with class probabilities
- B) Regression tasks
- C) Computing bounding-box class labels
- D) Selecting positive and negative pairs

Your answer: ~~A~~ ❌ → **B (Regression)**
> Classification uses cross-entropy (it works on a probability distribution). MSE measures squared distance between two real numbers, which is what you want when predicting a *value* — house price, age, depth — not a class.

**14. In object detection, mean Average Precision (mAP) is computed by:**
- A) Averaging the IoU across all images
- B) Averaging the Average Precision (area under the precision–recall curve) across classes
- C) Counting the correctly classified pixels
- D) Counting the number of bounding boxes per image

Your answer: ~~A~~ ❌ → **B**
> "mAP" decomposes as: AP (per class, area under the precision-recall curve) → mean over classes. IoU is used to decide which detections count as TP, but the metric itself is averaged precision, not averaged IoU.
